分子动理论内容

  1. 常识： 物质由分子组成
    1. 物理上的分子与化学上分子、原子、离子等微粒的区别所在
    2. 分子的构成: 原子
    3. 原子的构成: 质子 中子 原子核 他们的电性? 如何记忆? 原子不带电的原因?
  2. 常识: 分子无规则运动
    1. 扩散现象: 举例 煤炭 铅块 墨水 为什么鸡尾酒不扩散 生产半导体高温掺入(为什么高温)
    2. 布朗运动: 足够小微粒: 花粉 炭粒 做无规则运动剧烈程度影响因素: 微粒大小(为什么?) 温度 (举例? 夏天冬天的垃圾) 分子热运动的引出
    3. 布朗运动观察到的不是分子,为什么能表征微观上的分子? 判别相关说法正误 布朗运动引起了人们研究兴趣,分析了一通,得到分子无规则运动
    4. 热运动: 为什么是这个名字
  3. 分子间作用力
    1. 常识: 注射器压缩与拉开 固体 液体// 水与酒精混合,体积减少//分子之间有间隙
    2. 从化学角度解释为什么 气体分子间距大 22.4L/mol 水 0.018L/mol 显而易见
    3. 分子间作用力:
      1. 如何感受 拉伸,压缩物体 注射器拉伸与压缩
      2. 与万有引力的不同
      3. 产生原因 (结合化学 范德华力)
      4. 分子间作用力曲线: 正方向的定义 大小增减原因 引力斥力分别如何变化的
      5. 不能反映气体分子
  4. 总结 分子动理论包括以上三条

分子运动速率

  1. 假想气体分子运动状态: 匀速直线运动

    3. 不同温度的速率分布曲线
    4. 定义分子平均速率(分子平均动能)
  2. 气体压强产生原理:
    1. 分子的弹性碰撞 频繁 连续
    2. 压力&面积 压强&平均速率速度&分子数密度

分子能量

  1. 分子动能:
    1. 分子平均速度->分子平均动能的定义
    2. 分子平均动能只与温度有关,相同温度的不同气体(不同质量的气体分子)有相同的平均分子动能
    3. 推得 分子平均速度
    4. 宏观运动不改变分子动能
  2. 分子势能:

  3. 内能

0702 上课内容

热力学定律与能量守恒定律

  1. 能量守恒定律

    1. 能量不会凭空增加， 不会凭空消失 只会从一种形式转化成为另一种形式 或从一个物体转移到另一个物体。在转化和转移的过程中 能量的总量保持不变
    2. 第一类永动机不可能实现

  2. 热力学定律

    1. 第一定律
      1. 改变物体内能的两种方式 做功 热传递
      2. 热力学系统内能的增量等于外界向他传递的热量和外界对他做功的总和
      3. 符号表达 正负号
    2. 第二定律
      1. 卡劳修斯 热量不能自发从低温物体转移到高温物体
      2. 开尔文 不可能从单一热库吸收能量，使之完全变成功，而不产生其他影响 或 第二类永动机是不可能的
      3. 熵的解释 孤立系统的总熵不会减少
      4. 微观 一切自发过程总是沿着分子热运动无序性增大的方向进行

原子结构和波粒二象性

  1. 黑体 黑体辐射 能量子 普朗克常数
    1. 微观世界与宏观世界最显著的差异 量子化 连续化
    2. 能量子公式
    3. 普朗克常量 数值 单位
  2. 光电效应
    1. 带负电的锌板 受到紫外线照射之后 所带的负电荷减少
    2. 光电效应：光使金属中的电子逸出 逸出的电子成为光电子
    3. 光电效应 性质
      1. 截止频率 入射光的频率太小没有光电子逸出
      2. 饱和电流 电压增大 电流增幅逐渐减小； 光频率越高 饱和电流越大； 同种光 光强越大饱和电流越大
      3. 遏止电压 光电子有一定的初速度 计算公式
      4. 瞬时性
    4. 光电效应的原理
      1. 逸出功 电子脱离金属材料的能量
      2. 光由光子组成 爱因斯坦 光有能量子组成 称为光子 计算公式
      3. 计算公式
        1. 最大初动能
        2. 截止频率
        3. 遏制电压
  3. 康普顿效应 光子有动量
